My Work

Retinal Disease Classification using EfficientNet-B3

Retinal Disease Classification using
EfficientNet-B3

This project focuses on classifying retinal images to detect eye conditions like Normal, Cataract, Glaucoma, and Diabetic Retinopathy. Using the EfficientNet model and transfer learning, we fine-tuned pre-trained weights on a retinal image dataset. Image preprocessing and augmentation improved data quality, and the model showed promising results in accuracy and precision. This approach aims to assist healthcare professionals in diagnosing retinal diseases effectively.

Voice Enabled Form Filling Using Hidden Markov Model

Voice Enabled Form Filling Using
Hidden Markov Model

Our project focuses on automating the form-filling process using speech recognition technology to simplify tasks like job applications and internship submissions. The system analyzes a user's voice, converts speech into text, and seamlessly fills out forms, reducing the challenges of manual typing. This solution leverages a machine learning model based on the Hidden Markov Model, with Mel Frequency Cepstral Coefficients (MFCCs) as the preprocessing technique, widely recognized in speech recognition. The approach aims to enhance efficiency and accessibility in environments where voice-enabled automation is essential.

Paper Publication on HMM-Based Voice Form Filling

Paper Publication on HMM-Based
Voice Form Filling

This paper is published based on the project Voice Enabled Form Filling Using Hidden Markov Models, which was presented at IACC 2023. The project, developed in collaboration with my teammates, focuses on automating the form-filling process using speech recognition technology. The paper delves into the methodology, including the use of Hidden Markov Models and MFCC, to accurately transcribe speech into text for efficient data entry. This work highlights the potential of voice-enabled systems to streamline tasks and improve accessibility.
